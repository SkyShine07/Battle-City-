# 一，游戏概述

## 1.1 核心理念

《坦克大战：复兴》是一款基于经典FC《坦克大战》核心玩法全面重制的**3D俯视角**合作射击游戏。在保留经典坦克操作与基地防御灵魂的同时，引入现代游戏设计理念，通过技能系统、Buff/debuff系统、动态关卡等创新元素，为玩家带来既有怀旧情怀又有全新体验的战略射击游戏。

## 1.2 关键操作

- **经典操作，现代体验**：保留简单直观的坦克操控，加入丰富的策略维度。
- **深度合作**：多人合作，强调团队配合与职业互补。
- **动态战场**：可破坏环境、随机事件、多重目标，每局体验都独一无二。
- **视觉盛宴**：3D高清像素风格，华丽的技能特效与动态场景。

## 1.3 目标平台

PC (Steam)。

# 二，游戏机制

## 1，核心目标

保护位于基地中央的 鹰徽（老鹰） 不被敌方坦克摧毁，同时消灭所有出现的敌方坦克。

## 2. 基本规则

2.1 **生命规则**
   · 玩家：初始拥有一定数量的生命（通常为3条）。坦克被击中即损失一条生命。
   · 基地：鹰徽被任何子弹击中，游戏立即失败。
   · 敌方：无生命值概念，击中即毁。
2.2 **胜负条件**
   · 胜利：在玩家生命耗尽前，成功消灭关卡要求的所有敌方坦克。
   · 失败：
     · 玩家所有生命耗尽。
     · 基地鹰徽被摧毁。
2.3 **关卡推进**
   · 游戏由多个关卡组成。
   · 完成一关后，自动进入下一关，难度逐渐提升。


# 三，核心系统设计

## 2.1 基础操作与控制

- W，A,S,D：坦克八方向移动。
- 鼠标左键：开火。
- 鼠标滚轮：切换武器弹种。
- 自适应地形：不同的地表影响移动速度和操控性。

- 基础生成逻辑：仅红色闪光目标坦克被摧毁后触发道具掉落，70%概率掉落1个道具，30%概率无掉落；地形专属道具与核心道具可同时掉落（概率独立计算）。
    
- 概率修正：每通关3个关卡，核心道具总掉落概率提升5%（上限10%），新增道具同步提升，平滑难度曲线。
    
- 拾取交互：玩家触碰道具立即触发效果，伴随“叮”声音效（参考音效配置表SFX_012）；敌方仅高手难度可拾取，优先选择高威胁道具（铲子＞秒表＞炸弹）。
    
- 刷新限制：同一地形区域，新增道具两次掉落间隔≥50秒；道具生成后以0.5秒/周期闪烁，60秒内未拾取自动消失。
    
- 音效特效适配：每个道具对应专属音效与视觉特效，参数参考《音效与特效配置表》；AI生图提示词保留英文格式用于素材生成，风格统一贴合游戏科幻战斗基调。

##  AI 行为树 开发关键注意事项

1. **参数可配置化**：将所有触发条件（如距离阈值、概率、血量百分比）写入配置表，支持后期调优（无需修改代码）。
2. **性能优化**
    
    - 集群 AI 采用 **批处理检测**，避免每辆坦克单独检测视野（降低 CPU 消耗）；
    - 远距离敌人（>15 格）暂停精细行为逻辑，仅保留巡逻节点。
    
3. **与地形联动**：所有行为节点需加入地形判定逻辑，例如冰面移动时调整转向延迟参数，草地移动时提升隐蔽优先级。
4. **难度梯度适配**：
    - 新手难度：降低精英坦克出现概率，AI 预判射击精度下调 50%；
    - 高手难度：开启集群协同攻击，BOSS 召唤频率提升。

## ✅ 核心设计说明

1. **视觉奖励适配 UE3D 开发**：所有 3D 特效（残影、波纹、光环等）均基于 Niagara 粒子系统设计，可直接绑定坦克 / 技能蓝图，粒子数≤200，无性能压力。
2. **解锁进度可追溯**：UE Save Game 系统存储解锁进度，跨平台同步，卸载后重新安装可恢复，支持在游戏内 “成就面板” 查看实时进度。
3. **无平衡破坏**：所有奖励仅优化视觉表现，不提升伤害、防御、移速等属性，完全贴合 FC 原版 “纯操作 + 策略” 的核心乐趣。
4. **联机适配**：双人联机专属奖励需双方均达成条件方可解锁，鼓励协作，避免 “单方面努力” 无法解锁的情况。
5. **个性化展示**：解锁的称号可在主界面、战斗加载界面、击杀弹窗中自由选择展示，3D 视觉特效自动跟随生效，满足玩家个性化需求。

## 设计核心原则

### 视觉规范补充

1. **字体体系**：
    
    - 标题 / 核心数据：方正粗黑宋（3D 化处理，描边 1px），字号≥24px；
    - 次要文字 / 提示：微软雅黑（扁平化），字号 14-18px；
    - 称号 / 奖励：汉仪鎏金体（3D 鎏金效果），字号≥32px。
    
2. **特效限制**：所有 UI 粒子特效粒子数≤200，适配 UE Niagara 系统，4K 60fps 无性能损耗；
3. **适配性**：支持窗口 / 全屏切换，UI 元素自动缩放，键鼠 / 手柄操作均适配（手柄按键提示可在设置中切换显示）；
4. **无障碍设计**：高对比度模式（可切换纯文字高对比样式），重要提示（如基地受击）附带震动反馈（PC 端手柄）/ 屏幕闪烁。
